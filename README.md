# 👋 Real-Time Hand Tracking & Finger Counting

A very simple and lightweight Python program that uses your webcam to detect hand landmarks and count fingers in real-time.

## ✨ Features
* **Real-Time Tracking:** Processes your webcam feed instantly with minimal lag.
* **Accurate Finger Counting:** Uses advanced landmark detection to calculate which fingers are raised.
* **Lightweight:** Built on a simple stack using only three main libraries.

## 🛠️ Tech Stack
* **Python** (Recommended: v3.11 or v3.12 for optimal compatibility)
* **OpenCV** (`opencv-python`) for camera handling and image processing.
* **MediaPipe** (`mediapipe`) for the core machine learning hand-tracking models.
* **NumPy** (`numpy`) for array and matrix operations.
